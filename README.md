# Restaurant Website

This repository contains the final project of the course **"HTML, CSS, and JavaScript for Web Developers (Coursera)"**, focused on building a complete restaurant website.

The project integrates the knowledge and skills acquired throughout the course, including HTML structure, CSS styling, and JavaScript-based interactivity.

The implementations from **Modules 2 to 4** are organized in separate repositories, reflecting the incremental development approach followed during the course.
